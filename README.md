# Welcome to Tomo Graph!

Tomodachi Life has taken over my life recently, and I love updating my friends about my island. The relationships are whacky and fun to display, but with so many different miis, I realized just how inconvenient it was for Tomodachi to only show each character's relationships at a time, rather than one huge web of relationships between all the characters at once.

This application was created for users like me who want a way to display their island relationships in one easy image. Enjoy!

## How to Run

1) Make sure you are in the parent folder (Tomo-Graph)
2) Type "npm run dev" into terminal
